# Status

Display the task status:

:::note

A TUI which quto-refreshes every N seconds.

:::

```bash
$ osapi client task status

  ╭─────────────────────────────────────────────╮
  │                                             │
  │ Task Status                                 │
  │                                             │
  │ Currently processing 3 tasks.               │
  │                                             │
  │ Last Updated: Sun, 29 Sep 2024 22:13:53 PDT │
  │                                             │
  │ Press 'q' to quit                           │
  │                                             │
  ╰─────────────────────────────────────────────╯
```
