# Status

Show the status of the queue:

:::note

A TUI which quto-refreshes every N seconds.

:::

```bash
$ osapi client queue status

  ╭─────────────────────────────────────────────╮
  │                                             │
  │ Queue Status                                │
  │                                             │
  │ Queue is currently processing 3 tasks.      │
  │                                             │
  │ Last Updated: Sun, 29 Sep 2024 22:13:53 PDT │
  │                                             │
  │ Press 'q' to quit                           │
  │                                             │
  ╰─────────────────────────────────────────────╯
```
