# Start

To start the server:

```bash
$ osapi queue worker start
1:15PM INF queue configuration debug=false database.driver_name=sqlite database.data_source_name="file:database.db?_journal=WAL&_timeout=5000&_fk=true" database.max_open_conns=1 database.max_idle_conns=1 queue.poll_interval.seconds=30
```
